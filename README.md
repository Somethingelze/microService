# microService
Создать лаконичный микросервис на Java Spring Boot с JWT-аутентификацией, ролевым доступом и CRUD-функционалом для управления пользователями и заказами.

Проект предназначен для демонстрации:

архитектуры REST API
Spring Security + JWT
работы с PostgreSQL 17
миграций БД с Liquibase
Docker / Docker Compose
Unit тестирования
